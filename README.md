# Optimizing Store Performance Through Data-Driven Insights

## Project Overview
This project analyses store operational and economic data to identify factors that influence business performance. Using exploratory data analysis and machine learning techniques, the project uncovers relationships between key variables and develops predictive models to support data-driven decision-making.

## Business Problem
Retail organisations need to understand how operational and economic factors impact performance. This project investigates variables such as temperature, fuel prices, promotional markdowns, CPI, and unemployment rates to identify patterns that can support business planning and optimisation.

## Key Analysis Performed
- Explored store feature data and data quality
- Converted date fields into appropriate formats
- Encoded holiday indicators
- Analysed relationships between numerical variables
- Generated correlation heatmaps and visualisations
- Built regression models using operational and economic indicators
- Evaluated model performance using standard regression metrics

## Dataset
The dataset contains store operational features including temperature, fuel prices, promotional markdown activity, CPI, unemployment rates, holiday indicators, and store characteristics.
